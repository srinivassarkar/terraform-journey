* **Declare variable** in `variables.tf`:

```hcl

  type    = string
  default = "world"
}
```

* **Use in resource**:



## 4ï¸âƒ£ Locals

* Combine multiple variables:

```hcl

  intro = "Hello, I am ${var.name}. I am an ${var.job}."
}

  filename = "usingLocals.txt"
  content  = local.intro
}
```
